github相关网址打不开的解决办法

问题

常常因为网络环境，遇到 github.com、github.io、raw.githubusercontent.com 等网址打不开，比如：博客用 raw.githubusercontent.com 图床，出现图片裂开等情况。

解决方案

找一个域名解析的网站（比如：https://www.ipaddress.com/site/github.com），把地址输进去获取 IP，再配置 hosts。edge 可使用 edge://net-internals/#dns 再清空下 host cache，同理 chrome 更换为 chrome://net-internals/#dns 来清空缓存。
